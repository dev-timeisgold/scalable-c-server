#  High-Performance TCP Server
> 리눅스 환경에서 대용량 트래픽 처리를 목적으로 시스템의 한계를 극복해나가는 고성능 에코 서버 프로젝트입니다.

##  현재 진행 상황
- [x] **Phase 1:** 리눅스 시스템 콜을 이용한 기본 TCP 서버 소켓 기틀 마련 (진행 중)
- [ ] **Phase 2:** 다중 접속 처리를 위한 멀티 쓰레드 아키텍처 도입
- [ ] **Phase 3:** `epoll` 기반 이벤트 루프 최적화로 1만 명 동시 접속 달성

##  Phase 1 구현 핵심 (현재 상태)
현재 리눅스 커널의 네트워크 시스템 콜을 사용하여 서버의 가장 기초적인 파이프라인을 구축했습니다.

1. `socket()`: IPv4 주소체계와 TCP 프로토콜을 사용하는 가상 파일 디스크립터 생성 완료.
2. `setsockopt()`: 서버 재부팅 시 포트가 잠기는 에러를 방지하기 위해 `SO_REUSEADDR` 옵션 적용.
3. `bind()`: 서버 소켓에 특정 포트(8080)와 주소를 명확하게 바인딩 완료.
